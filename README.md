# Ecuador Localization - Odoo 17.0

Módulos de localización para Ecuador compatibles con Odoo 17.0.

## Módulos:
- **l10n_ec_sri**: Configuración SRI
- **l10n_ec_einvoice**: Facturación electrónica  
- **l10n_ec_withholding**: Retenciones

Este repositorio está configurado como submódulo para Odoo.sh.
